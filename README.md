# 2D Graphics Editor

## Project Description
This project is a simple 2D Graphics Editor developed in C using a 2D character array. The editor allows users to draw different geometric shapes on a text-based canvas.

## Features
- Draw Rectangle
- Draw Line
- Draw Circle
- Draw Triangle
- Clear Area
- Display Canvas

## Technologies Used
- C Programming Language
- 2D Character Array

## How to Run
1. Compile the program.
2. Run the executable.
3. Select options from the menu.
4. Draw shapes and display the canvas.
